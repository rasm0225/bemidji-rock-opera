# Audio Files Directory

Place your 48 audio files in this directory.

## Naming Convention

Your audio files should be named to match the paths in `script.js`:

```
01-opening-overture.mp3
02-welcome-to-the-northwoods.mp3
03-lake-of-shadows.mp3
04-babes-lament.mp3
... (and so on through 48)
```

## Format

- **Recommended:** MP3 format (192-320 kbps)
- **Also supported:** OGG, WAV, M4A
